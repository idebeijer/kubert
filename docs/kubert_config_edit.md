## kubert config edit

Edit the kubert config file in vim

### Synopsis

Open the kubert config file in vim editor for editing.

```
kubert config edit [flags]
```

### Options

```
      --create   create a new config file at $HOME/.config/kubert/config.yaml with default settings
  -h, --help     help for edit
```

### Options inherited from parent commands

```
      --config string   config file (default is $HOME/.config/kubert/config.yaml, can be overridden by KUBERT_CONFIG)
      --debug           debug mode
```
